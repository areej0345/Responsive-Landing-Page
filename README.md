## 📌 About

**Lumino** is a fictional SaaS product landing page designed as a frontend practice project. It showcases a polished UI with smooth layout, elegant typography, and mobile-first responsiveness — built entirely from scratch without any CSS framework.

> *"Illuminate your entire workflow"* — Lumino's hero tagline

---

## ✨ Features

- 📱 **Fully Responsive** — works seamlessly on mobile, tablet, and desktop
- 🍔 **Hamburger Menu** — mobile navigation with smooth open/close toggle
- 🎨 **Custom Typography** — uses Google Fonts (Playfair Display + DM Sans)
- ⚡ **Hero Section** — with badge, headline, sub-text, and CTA buttons
- 🧩 **Features Section** — 3-column card grid highlighting product pillars
- 🦶 **Footer** — with logo and navigation links
- 🚀 **No dependencies** — pure vanilla HTML/CSS/JS

## 🚀 Getting Started

No setup needed — just open in browser!

```bash
# Clone the repo
git clone https://github.com/areej0345/Responsive-Landing-Page.git

# Open in browser
cd Responsive-Landing-Page
open index.html
```

Or simply drag and drop `index.html` into any browser.
